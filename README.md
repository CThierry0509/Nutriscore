# Clean Food Score MVP - Zero-Budget Launch Package

This folder contains a working no-code/low-code MVP for Clean Food Score.

## What you have

- `index.html` - a ready-to-open website with product search and A-E score cards.
- `products.csv` - your starter product database, importable into Google Sheets.
- `products.json` - the same database for web use.
- `scoring_model.md` - the simple scoring rules and legal cautions.

## How to launch in 15 minutes for $0

### Option A - fastest local demo
1. Double-click `index.html`.
2. Search for "apple juice", "wine", "cereal", or "McDonald".
3. Show it to prospects or record a demo video.

### Option B - publish free with GitHub Pages
1. Create a free GitHub account.
2. Create a new public repository called `clean-food-score`.
3. Upload `index.html`, `products.csv`, `products.json`, and `scoring_model.md`.
4. Go to Settings > Pages > Deploy from branch > main.
5. Your site will be live at `https://yourusername.github.io/clean-food-score/`.

### Option C - publish free with Netlify
1. Create a free Netlify account.
2. Drag this entire folder into Netlify Drop.
3. Netlify gives you a public URL.

## Important legal/credibility rule

Do NOT say a specific product contains arsenic, lead, pesticides, etc. unless you have product-specific lab data.
Use:
- "category concern"
- "disclosure gap"
- "requires verification"
